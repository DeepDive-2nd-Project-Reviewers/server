## Project Description
 도서/영화 정보를 제공하고 리뷰를 작성해 감상평을 사용자들과 자유롭게 나누는 문화 커뮤니티 플랫폼입니다.

### [REST API 문서](http://3.36.118.226:8080/swagger-ui/index.html)

### 기능
- **User**
  - 회원가입
  - 로그인
  - 로그아웃
  - Google OAuth 로그인
- **Contents**
  - 영화/책 정보 조회
  - 영화/책 정보 생성
  - 영화/책 정보 수정
  - 영화/책 정보 삭제
- **Heart**
  - 콘텐츠 좋아요
  - 리뷰 좋아요
- **Review**
  - 리뷰 작성
  - 리뷰 조회
  - 리뷰 수정
  - 리뷰 삭제
- **Comment**
  - 리뷰에 댓글 생성
  - 리뷰별 댓글 조회
  - 댓글 수정
  - 댓글 삭제
